## Volumes

### В данной директории создаются тома при запуске приложения через docker compose:

- `/postgres`: Для хранения данных PostgreSQL.
- `/rabbitmq`: Для хранения данных RabbitMQ.
- `/redis`: Для хранения данных Redis.
- `/s3`: Для хранения данных MinIO (S3-совместимого хранилища).
- `/es`: Для хранения данных Elasticsearch.
