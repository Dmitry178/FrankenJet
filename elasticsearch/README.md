# Elasticsearch

Эта директория содержит файлы, связанные с настройкой Elasticsearch в проекте.

## Dockerfile

Файл `Dockerfile` используется для создания кастомного образа Elasticsearch с установленным плагином `analysis-icu`.

Плагин `analysis-icu` необходим для расширенного анализа текста, включая нормализацию символов (например, 'ё' -> 'е'), что улучшает качество поиска.

Этот кастомный образ используется в `docker-compose-full.yml` для запуска Elasticsearch с требуемым плагином внутри Docker-сети проекта.
